# cloud-sales-analytics-dashboard
# 📊 Cloud Sales Analytics Dashboard (GCP End-to-End Pipeline)

## 🚀 Project Overview

This project is an **end-to-end event-driven data engineering pipeline** built using Google Cloud Platform (GCP). It automates the process of ingesting sales CSV files, processing them in real time, storing them in BigQuery, and visualizing insights in Looker Studio.

The system is fully automated using **GitHub Actions + Cloud Storage + Eventarc + Cloud Run Functions + BigQuery + Looker Studio**.
